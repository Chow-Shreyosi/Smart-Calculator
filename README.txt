Smart Calculator

Overview

Smart Calculator is a Python-based calculator that performs basic mathematical operations through a menu-driven interface.

Features

- Addition
- Subtraction
- Multiplication
- Division
- Calculation History
- Input Validation
- User-Friendly Menu

Technologies Used

- Python 3

Project Structure

Smart-Calculator/
│
├── calculator.py
├── README.md
└── screenshots/

How to Run

1. Open the project in Pydroid 3 or any Python IDE.
2. Run "calculator.py".
3. Select an operation from the menu.
4. Enter the required numbers.
5. View the result.
